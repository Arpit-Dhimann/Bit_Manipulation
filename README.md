# 🚀 Java Operators & Bit Manipulation

Welcome to my Java logical foundation repository! This project is dedicated to understanding how computers process data at the bit level and mastering operator logic. 

Here, I explore fundamental programming concepts ranging from base conversions to advanced bitwise operations.

---

## ✅ Current Progress: Bit Manipulation

Inside this repository, you will find clean, modular implementations of core bit manipulation techniques:

- [x] **Get Bit** (`GetBit.java`): Logic to check if a specific bit at a given position is 0 or 1.
- [x] **Set Bit** (`SetBit.java`): Logic to set a bit to 1 at a specific position using bitwise OR.
- [x] **Clear Bit** (`ClearBit.java`): Logic to set a bit to 0 at a specific position using bitwise AND and NOT.
- [x] **Update Bit** (`UpdateBit.java`): A dynamic utility to either set or clear a bit based on user input.

---

## 🛠 Tech Stack & Concepts Covered

- **Language:** Java (JDK 17+)
- **Key Concepts:**
    - **Bitwise Operators:** `&` (AND), `|` (OR), `~` (NOT), `<<` (Left Shift).
    - **Binary System:** Understanding base-2 representation.
    - **Masking:** Creating and using bitmasks for efficient data manipulation.
- **Tools:** VS Code, Git & GitHub.
